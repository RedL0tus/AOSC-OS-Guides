### 添加用户

这时候其实你已经有一个完全可以工作的 AOSC OS 了。

但是你还没办法使用它，因为你还没有创建你自己的账户。

你需要一个账户来使用这个系统，而下面这个命令可以为你做到这点：

```
# useradd -m -G wheel -s /bin/bash <用户名>
```

解释：

* `useradd`：添加用户
  * `-m`：创建家目录（也就是个人目录）

  * `-G wheel`：添加到 wheel 组
  * `-s /bin/bash`：设置默认 shell，你也可以选择别的，如果你知道你在做什么的话。
  * 最后就是你自己喜欢的用户名啦！

之后还需要设置一下密码：

```
# passwd aosc
```

如果你想的话，还可以给拥有最高权限的 root 先生设置一下密码：

```
# passwd root
```
